---

aonzoraconv is a tool/library to convert Shift_JIS <-> Unicode texts in Aozora Bunko Format.

aozoraconvは青空文庫形式のテキストをShift_JISからUnicodeに変換したり、その逆を行ったりするためのツール兼ライブラリです。
